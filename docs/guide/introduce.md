## 安装

使用 npm 安装

```bash
npm install suda
```

使用 yarn 安装

```bash
yarn add suda
```

## 示例

```ts
import ReactDOM from 'react-dom/client';
import { Button } from 'suda';

ReactDOM.createRoot(document.getElementById('root')!).render(<Button />);
```
