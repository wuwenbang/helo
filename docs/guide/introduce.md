## 介绍

Suda 是一个易用的轻量级的 React UI 组件库。

## 安装

使用 npm 安装

```bash
npm install suda
```

使用 yarn 安装

```bash
yarn add suda
```

## 示例

```ts
import ReactDOM from 'react-dom/client';
import { Button } from 'suda';

ReactDOM.createRoot(document.getElementById('root')!).render(<Button />);
```
