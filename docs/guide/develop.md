# Suda

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn dev
```

Build documentation,

```bash
$ yarn docs:build
```

Run test,

```bash
$ yarn test
```

Build library via `father-build`,

```bash
$ yarn build
```
